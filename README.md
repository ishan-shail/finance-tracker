## Project Description
This is a fully functional, client-side financial analytics web application built using **Vanilla JavaScript**, **HTML**, and **CSS**.  
The application helps users track income and expenses, analyze financial behavior, and visualize expense distribution using dynamic DOM manipulation without any frontend frameworks.

---

## Problem Statement
Students and individuals often fail to understand their financial habits due to lack of structured tracking and analysis tools.  
This project solves that problem by providing an interactive finance tracker that calculates income, expenses, balance, savings rate, and category-wise expense distribution in real time using pure JavaScript and DOM-based rendering.

---

## Features Implemented
- Add and delete income/expense transactions
- Category-based expense tracking
- Real-time calculation of:
  - Total income
  - Total expense
  - Net balance
  - Savings percentage
- Expense distribution bar graph (without libraries)
- Filter transactions by type (All / Income / Expense)
- Persistent data storage using LocalStorage
- Input validation and error handling
- Dynamic empty states and conditional rendering
- Responsive and tech-themed UI
